# FlickUAC
透過在註冊表中加入 RunAsInvoker 相容性層級，強制讓特定文件以呼叫者權限執行，從而避開使用者帳戶控制 UAC (User Account Control) 的權限提升請求。

## 核心功能

### 1. 現代化清單管理
* **資源自動解析**：程式會自動解析並提取路徑對應的執行檔圖示，提供直觀的視覺標識，方便快速辨識項目。
* **響應式選取邏輯**：重構後的選取機制確保在大規模資料下仍能保持流暢的操作反饋。
* **高效能架構**：基於 .NET 10.0 開發，確保在處理系統底層註冊表時具備極高的執行效率。

### 2. 進階批量操作
* **多重項目選取**：支援標準的 Extended Selection 模式，可透過滑鼠拖曳、Shift 或 Ctrl 鍵同時選取多個目標項目。
* **同步處理能力**：支援針對多組註冊表鍵值進行同步刪除或檔案位置檢索。

## 功能按鈕操作說明

### 1. 加入項目
* **作用**：手動建立新的 UAC 繞過註冊表項。
* **用法**：點擊「加入」按鈕，在彈出的對話框中選取欲執行之 `.exe` 檔案或輸入特定指令路徑。確認後程式將在對應的註冊表路徑下建立對應的註冊表值。

### 2. 批量刪除
* **作用**：從系統註冊表中移除選定項目的繞過設定。
* **用法**：在清單中選取一或多個項目後點擊「刪除」。程式將呼叫註冊表 API 執行刪除作業，並同步將其從 UI 清單中移除。

### 3. 批量查看詳細內容
* **作用**：檢視選定項目的詳細路徑、圖示資訊以及定位實體檔案。
* **用法**：
    * **內容預覽**：滑鼠停留在項目上會顯示完整路徑。
    * **檔案定位**：選取項目後點擊「位置」，程式將調用檔案總管，自動開啟資料夾並選中該實體檔案。

### 4. 語言切換
* **作用**：切換介面顯示語言。
* **用法**：點擊語言切換按鈕。程式將即時從 JSON 語系檔案重新映射介面字串，實現 Runtime 狀態下的介面語系更換，無需重新啟動程式。

### 5. 重新整理
* **作用**：重新掃描註冊表並同步最新的資料狀態。
* **用法**：點擊「重新整理」按鈕。程式會重新遍歷目標註冊表路徑並重新載入資料，確保 UI 顯示與系統實際狀態完全一致。

## 技術規格
* 執行平台：Windows x64
* 目標框架：.NET 10.0
* 使用者介面：WPF (Windows Presentation Foundation)

## 部署與下載說明

為確保程式能正常啟動，請根據您的系統環境選擇對應的版本：

### 獨立版 - 推薦
* **適用對象**：一般使用者，或不確定電腦是否安裝 .NET 10 環境的系統。
* **說明**：此版本採用 Self-contained 部署，已包含 .NET 10 必要組件，檔案體積較大但具備最佳相容性。

### 標準版
* **適用對象**：已手動安裝 .NET 10 Runtime 的使用者。
* **說明**：檔案體積極小，依賴系統已存在的 .NET 環境，若環境版本不符則無法啟動。

## 操作安全性提示
本工具涉及對系統註冊表 `HKEY_CURRENT_USER` 路徑的寫入與刪除操作。在執行「批量刪除」或「加入項目」前，請務必確認操作目標符合您的預期。
